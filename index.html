ArchViz 3D

Blueprint Viewer

Upload 2D floor plans and explore them as interactive 3D building models.

How to Launch

1
Open a terminal in the project root and run npm run install:all to install dependencies.
2
Run npm run dev to start both the frontend and backend servers.
3
Open the app in your browser at localhost:5173.
